# Dozee Smart Healthcare Command Center

## Project Overview
This interactive prototype demonstrates a strategic digital transformation intervention for healthcare facilities, utilizing Dozee's contactless monitoring infrastructure. 

## Strategic Problem Resolution
This dashboard operationalizes predictive intelligence to solve critical healthcare bottlenecks:
* **Reactive Monitoring**: Replaces manual 4-hourly spot checks with continuous, AI-driven Early Warning Scores (MEWS).
* **Resource Allocation**: Introduces Acuity-Based Dynamic Routing to direct nursing staff strictly to moderate and high-risk patients.
* **Fragmented Data**: Consolidates the real-time "Speed Layer" (vitals) and the historical "Batch Layer" (readmission risk) into a unified single pane of glass.
